Detailed Description

Device driver module for serial port communication.

This module provides the necessary functionality to communicate with a serial port (UART). It includes functions for initializing the serial port, sending and receiving data and handling interrupts. The module also contains definitions for relevant registers. On top of that it also includes the implementation of a queue used for communication.
